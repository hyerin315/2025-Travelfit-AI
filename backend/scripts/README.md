# 백엔드 스크립트

개발 및 운영에 유용한 스크립트 파일들입니다.

## 사용 가능한 스크립트

### setup_env.sh
환경 변수 설정을 위한 대화형 스크립트입니다.

```bash
# 사용법
cd backend
./scripts/setup_env.sh
```

이 스크립트는 `.env` 파일을 생성하고 Hugging Face API 토큰을 설정합니다.

## 향후 추가 예정

- `run_dev.sh` - 개발 서버 실행
- `run_tests.sh` - 테스트 실행
- `format_code.sh` - 코드 포맷팅
- `lint_code.sh` - 코드 린팅
